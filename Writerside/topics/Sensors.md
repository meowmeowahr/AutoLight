# Sensors

The two currently supported sensor types are VL53L0X via i2c (vl53l0x_i2c) and GPIO (gpio).

### Pros and Cons

|                      | GPIO | VL53L0X |
|----------------------|------|---------|
| Simplicity           | ✅    | ❌       |
| Versatility          | ✅    | ❌       |
| Ease of Wiring       | ❌    | ✅       |
| Reliability          | ✅    | ❌       |
| Distance Measurement | ❌    | ✅       |
